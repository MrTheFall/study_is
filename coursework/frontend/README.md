# Krusty Crab Frontend

Фронтенд приложение для системы управления рестораном "Красти Крабс".

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **React Router** - роутинг
- **Zustand** - управление состоянием
- **React Hook Form + Zod** - формы и валидация
- **Tailwind CSS** - стилизация
- **Axios** - HTTP клиент
- **OpenAPI Generator** - генерация API клиента из OpenAPI спецификации

## Установка

```bash
npm install --legacy-peer-deps
```

## Генерация API клиента

После изменения OpenAPI спецификации в `../backend/src/main/resources/openapi.yml`:

```bash
npm run generate-api
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
├── api/
│   ├── client.ts          # Настройка API клиента
│   └── generated/         # Сгенерированные типы и API клиенты
├── components/
│   ├── ui/                # Базовые UI компоненты
│   └── ProtectedRoute.tsx # Защищенный роут
├── pages/                 # Страницы приложения
├── store/                 # Zustand stores
├── lib/                   # Утилиты
└── App.tsx                # Главный компонент
```

## Функциональность

### Для клиентов:
- Регистрация и вход
- Просмотр меню
- Создание заказов
- Просмотр истории заказов
- Оставление отзывов

### Для сотрудников:
- Вход в систему
- Управление заказами (кухня)
- Управление инвентарем
- Просмотр смен

### Для управляющего:
- Управление сотрудниками
- Просмотр аналитики
- Управление сменами

## Переменные окружения

Создайте файл `.env` (опционально):

```env
VITE_API_PORT=13228
```

Frontend по умолчанию обращается к тому же хосту, с которого открыт сайт, на порт `VITE_API_PORT` (по умолчанию `13228`). Если нужно явно указать другой адрес (например, для SSR/CI), задайте `VITE_API_BASE_URL`.
